# TSM_CloudSys – Lab 4: Circuit Breaker on Kubernetes

This repository contains the implementation and report material for **Lab 4**
of the module **TSM_CloudSys (Cloud Services and Systems)**.

## Topic
Circuit Breaker pattern implemented with **NGINX Plus** on **Kubernetes**,
including fallback routing and active health checks.

## Technologies
- Kubernetes (Minikube)
- NGINX Plus
- Node.js (mock backend)
- ConfigMaps & Secrets

## Structure
- `k8s/` – Kubernetes manifests
- `report/` – Lab report (Markdown / PDF)
- `scripts/` – Helper commands
